# Proiect DAW – Agentie de Turism

Aplicatie web dezvoltata in PHP + MySQL.

Functionalitati:
- Autentificare / inregistrare
- CRUD rezervari
- Roluri utilizatori (admin / user)
- Admin Dashboard pentru administrare pachete turistice/rezervari/useri.
- Formular contact/ inregistrare/ login
- Optiuni de import/export 
- Statistici
